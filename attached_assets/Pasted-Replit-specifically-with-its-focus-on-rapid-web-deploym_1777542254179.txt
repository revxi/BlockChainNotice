### 🛠️ Step-by-Step Guide to Using this Prompt in Replit

To make sure you get the absolute best results, follow these setup steps inside Replit before pasting the prompt:

#### 1. Create the Repl with the Right Stack
You must set up the project environment to handle Next.js and Tailwind CSS (which is what makes this "clean" aesthetic possible).

*   Go to Replit and create a new Repl.
*   **Search for the Template:** `Next.js` (Choose the official Replit template). Replit will pre-configure the Node.js environment for you.

#### 2. Open the Code AI (Ask AI) Panel
Once your environment loads, locate the `Ask AI` button (usually in the bottom right corner or accessible via the chat interface). This is where you paste the prompt.

#### 3. Initial Configuration (If Prompt Fails)
If the prompt fails because it says the environment isn't set up for Tailwind, you can quickly run the configuration commands first. Type this into the **Console** (not the AI window):

```bash

npm install -D tailwindcss postcss autoprefixer

# Initialize Tailwind and PostCSS
npx tailwindcss init -p
```

Then, try pasting the prompt into the **Ask AI** panel again.

#### 4. Post-Generation Fine-Tuning
Once the AI generates the `pages/index.js` (or `index.tsx`) file and the associated components, you can manually fine-tune the aesthetics by editing the `tailwind.config.js` or `styles/globals.css` file to refine the soft gradients, whitespace, and font weights shown in the image reference.
